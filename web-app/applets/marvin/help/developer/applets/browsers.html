Browser compatibility of Marvin Applets

Marvin Applets are tested with several browsers on various platforms. This document shows the recommended browsers for each operating system.
  • Windows
  • Linux
  • OS X
    • 10.5 (Leopard)
    • 10.4 (Tiger)
    • 10.3 (Panther)
  • Solaris 10
  • Java - JavaScript communication
