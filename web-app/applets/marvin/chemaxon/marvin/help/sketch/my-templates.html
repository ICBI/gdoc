Adding new Templates
  • Creating and adding:
    1. Draw a structure.
    2. Select it.
    3. Drag the selection with the mouse and drop it to the My Templates window.
  • Options:
    • Select "Add New Items" in the Options menu if you want the new templates to be added at the end.
    • Select "Insert New Items" in the Options menu if you want the new template to be inserted at the mouse cursor.
Removing a Template
  • Right click for the pop-up menu and select "Remove".
 Template Properties
  • Right click for the pop-up menu and choose "Properties" to change a template property.
  • Set the abbreviation if you want to enable the access of your template using the keyboard.
  • By setting the rotation unit angle, you can make the template button rotatable.
Changing the number of columns
  • Choose "Set Number of Columns" in the Options menu.
